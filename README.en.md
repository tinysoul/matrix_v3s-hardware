# matrix_v3s hardware

#### Description
a simple allwinner V3s dev board designed by KiCad.

#### Usage

add [my_kicad_lib](https://github.com/matrixant/my_kicad_lib.git) to your kicad symbol and footprint lib path (Optional), then open the .pro file by KiCad.
